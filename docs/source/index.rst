

Oppia User Guide
============================================

Oppia is an open source learning platform with the goal of providing engaging content to learners. Anyone can create and share lessons with the aim of recreating a one-on-one tutoring scenario. 

We suggest getting  familiar with the :ref:`key terms used in Oppia <keyconcepts>` first.
